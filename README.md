# Course Allocation System

An automated system designed to streamline and manage the process of assigning academic courses to lecturers. This project helps university departments avoid schedule conflicts, balance teaching loads, and organize faculty schedules efficiently.

## 🚀 Features
* Lecturer Management: Profile creation and tracking of teaching preferences or specialties.
* Course Catalog: Database of available courses, credit units, and department requirements.
* Conflict Prevention: Automated checks to ensure lecturers are not double-booked.
* Dashboard View: Easy-to-read summary of which lecturer is teaching which course.

## 🛠️ Tech Stack
* Frontend: HTML, CSS, JavaScript,
* Backend:  PHP
* Database: MySQL (via XAMPP phpMyAdmin)

## 💻 How to Run the Project
1. Download or clone this repository as a `.zip` file.
2. Extract the folder and move it into your XAMPP installation directory: `C:\xampp\htdocs\`
3. Open the **XAMPP Control Panel** and start both **Apache** and **MySQL**.
4. Open your browser and go to `http://localhost/phpmyadmin/` to create a new database.
5. Import your project's `.sql` database file (if included) into phpMyAdmin.
6. Open a new tab in your browser and visit: `http://localhost/course-allocation-system/`
